1. 게임 개요

이 스도쿠 게임은 웹 브라우저에서 바로 즐길 수 있는 완전판 스도쿠입니다.
간단히 말해, 9×9 격자에 1~9까지 숫자를 채워 넣어 각 행, 열, 3×3 구역에 같은 숫자가 중복되지 않게 만드는 퍼즐 게임이죠.

이 게임은 단순 입력만 하는 것에서 끝나지 않고, 다양한 편의 기능과 UI 요소가 추가되어 스도쿠 초보자부터 고수까지 모두 즐길 수 있는 웹 게임으로 제작되었습니다.

2. 핵심 기능
① 난이도 선택

Easy / Normal / Hard로 난이도 선택 가능

난이도에 따라 빈 칸 수가 조절되어 게임 난이도가 달라집니다.

Easy: 35개 제거

Normal: 45개 제거

Hard: 55개 제거

② 자동 퍼즐 생성

클릭 한 번으로 새로운 퍼즐이 생성됩니다.

퍼즐은 무작위 배치 + 백트래킹 알고리즘으로 생성되며, 매번 다른 도전 제공

③ 힌트 기능

남은 빈 칸 중 하나를 자동으로 채워주는 힌트 버튼

초보자가 막힐 때 유용하게 활용 가능

④ 오답 체크 & 자동 하이라이트

입력한 숫자가 정답과 다르면 붉은색 배경으로 표시

“오답 자동 하이라이트” 옵션을 켜면, 잘못된 입력을 실시간으로 확인 가능

⑤ 타이머

게임 시작 버튼을 누르면 실시간 타이머 시작

일시정지, 리셋 기능도 제공

⑥ 저장/불러오기

현재 퍼즐 상태와 입력 상태, 타이머까지 브라우저 로컬 스토리지에 저장 가능

게임을 중간에 종료하고 나중에 이어서 플레이 가능

⑦ 숫자 입력 UI

하단에 숫자 버튼 1~9 및 지우기 버튼 제공

클릭 또는 키보드 입력으로 쉽게 숫자를 입력할 수 있음

⑧ 애니메이션

선택된 셀은 강조 표시

옵션에서 애니메이션 ON/OFF 가능
